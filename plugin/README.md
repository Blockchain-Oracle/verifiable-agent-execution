# plugin/ — the OpenClaw plugin source

This is THE OpenClaw plugin shipped by this repo. Captures every
tool call inside an agent session, encrypts the log, flushes it to
0G Storage, and mints an ERC-7857 iNFT receipt on AgenticID.

End users install the **built** version from npm:
[`@blockchainoracle/openclaw-verifiable-execution`](https://www.npmjs.com/package/@blockchainoracle/openclaw-verifiable-execution).
This folder is the workspace SOURCE that gets bundled into that tarball.

## Layout

```
plugin/
├── src/                    TypeScript source
│   ├── index.ts            Plugin entrypoint — registers hooks + slash commands
│   ├── SessionManager.ts   In-memory buffer of active sessions
│   ├── crypto.ts           AES-256-GCM envelope (matches packages/logger)
│   ├── keystore.ts         Per-tokenId key persistence (~/.openclaw/...)
│   ├── wallet.ts           Plugin's signing wallet (~/.openclaw/...)
│   ├── share-command.ts    The /share slash command
│   ├── transcript-parser.ts  Claude Code session.jsonl parser (Tier 1.5)
│   ├── config.ts           zod-validated runtime config
│   └── hash.ts             sha256 + canonical JSON
├── tests/                  vitest suites (171 tests)
├── dist/                   esbuild output (gitignored, npm-publish target)
├── openclaw.plugin.json    OpenClaw manifest — config schema
├── build.mjs               Bundler: src/ → dist/ (esbuild ESM)
└── package.json            Workspace pkg @verifiable-agent-execution/plugin
                            (npm name is different — see build.mjs)
```

## Common commands

```bash
# Type-check (uses extends ../tsconfig.json)
pnpm --filter @verifiable-agent-execution/plugin typecheck

# Run tests (171 tests, ~1s)
pnpm --filter @verifiable-agent-execution/plugin test

# Build the npm-publishable bundle
pnpm --filter @verifiable-agent-execution/plugin build
# → produces plugin/dist/{index.js, package.json, openclaw.plugin.json, README.md}

# Install the local build into your OpenClaw (for testing without publishing)
openclaw plugins install --link "$(pwd)/plugin/dist"
```

## Workspace vs npm-publish package

| | Name | Where |
|---|---|---|
| Workspace pkg (for `pnpm --filter`) | `@verifiable-agent-execution/plugin` | `plugin/package.json` |
| npm-publish pkg (what users install) | `@blockchainoracle/openclaw-verifiable-execution` | `plugin/dist/package.json` (generated by build.mjs) |

The npm-publish name is BAKED IN to `build.mjs`. Don't rename it — every
existing install in the wild points at it.

## How it fits together

```
plugin/src/index.ts
    ├── imports packages/chain-client (SessionAnchor — mints the iNFT)
    ├── imports packages/logger (SessionLogger — buffers + encrypts + uploads)
    ├── imports packages/tee-adapter (signing helpers)
    └── registers OpenClaw hooks:
          - after_tool_call    capture every tool entry
          - before_agent_finalize  parse Claude Code transcript
          - agent_end          flush + mint
          - inbound_claim      handle /share + /tokens commands
```
