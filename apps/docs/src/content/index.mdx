# AGENTSCAN — Etherscan for AI agents

**AGENTSCAN** turns every action your AI agent takes into a
cryptographically signed, on-chain-anchored receipt. Share one URL.
Anyone — no wallet, no login, no setup — can verify the agent did
exactly what it claimed.

It's an [OpenClaw](https://openclaw.ai) plugin that installs in one
command and anchors every session to the [0G](https://0g.ai) chain.

## Why?

Today, "did the agent really do that?" has no good answer. Logs can
be edited. Screenshots can be faked. A trace from the agent
operator's own server is a "trust me, bro."

AGENTSCAN gives you a public, tamper-proof primitive instead:

- **Cryptographically signed** — each tool call is signed inside the
  agent's TEE container before it ever leaves the box. The agent
  can't tamper with its own log.
- **On-chain anchored** — the session's content hash is minted as
  an ERC-7857 iNFT on 0G AgenticID. The hash is permanent and
  globally readable.
- **Encrypted by default** — the content lives on 0G Storage,
  encrypted with a key only you control. You choose who decrypts.
- **Verifiable cold** — anyone with the URL can confirm the chain
  anchor and storage existence even without the decryption key.
  When you share the key (via /agentscan_share), they decrypt locally.
