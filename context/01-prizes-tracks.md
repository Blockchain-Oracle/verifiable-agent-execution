## Submission requirements

1. **Basic project info**: Name + one-sentence description ≤30 words + summary of what it does / problem it solves / which 0G component(s) used
2. **GitHub repo**: Public, with substantial hackathon-period commits. Empty/placeholder repos disqualified.
3. **0G Integration proof**: 0G mainnet contract address + 0G Explorer link showing verifiable on-chain activity. At least one of: 0G Storage, 0G Compute, 0G Chain, Agent ID, Privacy/TEE features.
4. **Demo video**: ≤3 minutes. Must show core function + user flow + how 0G component is used. Slide-only or concept-only videos rejected. Public YouTube/Loom link required.
5. **README/Documentation**: English or Chinese. Must include: project overview, architecture diagram or technical description, which 0G modules are used + how, local deployment steps, test account / faucet notes.
6. **Public X post**: Project name + demo screenshot or clip + `#0GHackathon #BuildOn0G` + `@0G_labs @0g_CN @0g_Eco @HackQuest_`. Mandatory.
7. **Optional bonus**: Pitch deck, frontend demo link, user feedback screenshots, backend API docs, technical tutorial.

## Track 1 — Agentic Infrastructure & OpenClaw Lab (target track)

**Core Focus:** Developing the cognitive backbone and orchestration layers for autonomous intelligence.

**Technical Scope:** Building agent frameworks, specialized Skills, and data-processing pipelines.

**Priority Requirements (verbatim from listing):**
> "We encourage the integration of OpenClaw for orchestration and 0G Compute for model fine-tuning or inference. Projects should demonstrate the use of 0G Storage for state persistence and long-context memory."

**What 0G wants (from ETHGlobal OpenAgents parallel event, same sponsor, Apr 24–May 6):**

Track A — Best Agent Framework, Tooling & Core Extensions ($7,500 prize in that event):
- New OpenClaw modules for hierarchical planning, reflection loops, or multi-modal reasoning natively integrating 0G Compute's sealed inference
- Self-evolving agent framework that autonomously generates/tests/integrates new skills using persistent 0G Storage memory
- Modular "agent brain" library with swappable memory layers, LLM backends, decision engines
- No-code/low-code visual agent builder with one-click deployment to 0G Compute + Storage

Track B — Best Autonomous Agents, Swarms & iNFT Innovations ($7,500 prize):
- Personal "Digital Twin" agent with evolving memory via 0G Storage (KV for real-time state + Log for conversation/history)
- Specialist agent swarms collaborating via shared 0G Storage memory + 0G Compute inference
- iNFT-minted agents with encrypted intelligence on 0G Storage, dynamic upgrades, automatic royalty splits
